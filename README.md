# Sudoku-BruteForce
Sudoku BruteForce

Can be run without giving an input ( has a default board) or it can be used with a input like this:

python3 sudoku.py "........." "5.3.67..." "9..3421.." ".....4..." "..1...72." "..2.1...." ".3......9" ".8.1..2.." "...75.8.6"

it has checks to see if your input is correct
